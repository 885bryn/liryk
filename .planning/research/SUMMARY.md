# Project Research Summary

**Project:** Spotify Live Lyrics Web App
**Milestone:** v1.1 Web App Foundation and Theming
**Researched:** 2026-03-20

## Key Findings

- **Stack additions:** Use a web app entrypoint with Tailwind token theming and shadcn/ui primitives; preserve validated domain logic where possible.
- **Feature table stakes:** Browser shell, responsive layout, light/dark toggle, persistent theme, and polished lyric-state presentation.
- **Watch out for:** Installing shadcn/ui too late, treating theme as cosmetic-only, and carrying desktop-only assumptions into web runtime.

## Recommendation

Proceed with research-informed requirements and roadmap. Make shadcn/ui installation an explicit early implementation checkpoint in Phase 5, right after web scaffold and Tailwind base setup.
